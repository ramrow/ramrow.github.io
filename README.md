# ramrow.github.io

Personal portfolio site based on the Academic Pages Jekyll template.

## Local Preview

This site is a Jekyll project. To preview locally:

```powershell
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.

## Customize

Start with `_config.yml` for site-wide settings, then edit content in `_pages/`
and `_research/`.

Template source: https://github.com/academicpages/academicpages.github.io
